# Tmdb::Movies

A gem for easy access to themoviedb.org's api.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tmdb-movies'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tmdb-movies
